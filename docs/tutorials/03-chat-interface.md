# Chat Interface

Master the AI chat interface for document Q&A.

## Getting Started

1. Navigate to the Chat page
2. Select a document collection (or query all documents)
3. Type your question and press Enter

## Query Modes

### Document Mode

Query specific documents in your library. Best for:
- Factual questions
- Document summaries
- Extracting specific information

### Web Mode

Search the web for current information. Best for:
- Recent events
- General knowledge
- Cross-referencing

## Advanced Features

### Multi-turn Conversations

The chat maintains context across messages, allowing follow-up questions.

### Source Citations

Every response includes clickable citations to the source documents.

### Suggested Questions

After each response, see AI-generated follow-up questions.

## Query Tips

- Be specific: "What is the Q3 revenue?" vs "Tell me about revenue"
- Use document references: "According to the annual report..."
- Ask for comparisons: "Compare X and Y in the documents"

## Keyboard Shortcuts

- `Enter`: Send message
- `Shift+Enter`: New line
- `Esc`: Clear input
